MonetaryAmount

A Schema.org Data Type

Canonical URI: http://schema.org/MonetaryAmount

<SDOTYPE: 'Thing' expanded: False> -> <SDOTYPE: 'Intangible' expanded: False> -> <SDOTYPE: 'StructuredValue' expanded: False> -> <SDOTYPE: 'MonetaryAmount' expanded: False>

A monetary value or range. This type can be used to describe an amount of money such as $50 USD, or a range as in describing a bank account being suitable for a balance between £1,000 and £1,000,000 GBP, or the value of a salary, etc. It is recommended to use PriceSpecification Types to describe the price of an Offer, Invoice, etc.
