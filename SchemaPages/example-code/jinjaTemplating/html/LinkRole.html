LinkRole

A Schema.org Data Type

Canonical URI: http://schema.org/LinkRole
Note: This term is pending in the vocabulary.

<SDOTYPE: 'Thing' expanded: False> -> <SDOTYPE: 'Intangible' expanded: False> -> <SDOTYPE: 'Role' expanded: False> -> <SDOTYPE: 'LinkRole' expanded: False>

A Role that represents a Web link e.g. as expressed via the 'url' property. Its linkRelationship property can indicate URL-based and plain textual link types e.g. those in IANA link registry or others such as 'amphtml'. This structure provides a placeholder where details from HTML's link element can be represented outside of HTML, e.g. in JSON-LD feeds.
