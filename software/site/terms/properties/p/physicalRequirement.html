physicalRequirement

A Schema.org Property

This term is proposed for full integration into Schema.org, pending implementation feedback and adoption from applications and websites.
Thing > Property > physicalRequirement
  • Canonical URL: http://schema.org/physicalRequirement
  • Leave public feedback on this term 💬
  • Check for open issues.
A description of the types of physical activity associated with the job. Defined terms such as those in O*net may be used, but note that there is no way to specify the level of ability as well as its nature when using a defined term.
