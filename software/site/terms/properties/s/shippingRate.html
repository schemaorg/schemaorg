shippingRate

A Schema.org Property

This term is proposed for full integration into Schema.org, pending implementation feedback and adoption from applications and websites.
Thing > Property > shippingRate
  • Canonical URL: http://schema.org/shippingRate
  • Leave public feedback on this term 💬
  • Check for open issues.
The shipping rate is the cost of shipping to the specified destination. Typically, the maxValue and currency values (of the MonetaryAmount) are most appropriate.
Values expected to be one of these types
MonetaryAmount
Used on these types
OfferShippingDetails
ShippingRateSettings
