Schema.org core schema

This is an experimental RDFa 1.1 Lite representation of the schema.org schema, copied here for collaboration and feedback.

Thing 事物 The most generic type of item. 一切事物的基类
CreativeWork 创作品 The most generic kind of creative work, including books, movies, photographs, software programs, etc. 各种创造物的基类，包括书籍、电影、照片，软件等. Subclass of: Thing Source: rNews
WebPage 网页 A web page. Every web page is implicitly assumed to be declared to be of type WebPage, so the various properties about that webpage, such as <code>breadcrumb</code> may be used. We recommend explicit declaration if these properties are specified, but if they are found outside of an itemscope, they will be assumed to be about the page 基础网页类型，所有的互联网的页面，都是一个网页 Subclass of: CreativeWork
AboutPage 关于页 Web page type: About page. 一个网页的类型: 关于页面. Subclass of: WebPage

Place 地点 Entities that have a somewhat fixed, physical extension. 广泛意义上的一个地方 Subclass of: Thing
LocalBusiness 本地商业 A particular physical business or branch of an organization. Examples of LocalBusiness include a restaurant, a particular branch of a restaurant chain, a branch of a bank, a medical practice, a club, a bowling alley, etc. 服务于某一区域的（分支）机构，比如一个连锁餐馆的某一个分店，或者一个银行，俱乐部等. Subclass of: Organization Subclass of: Place
